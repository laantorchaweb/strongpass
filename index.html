Options

You can initialize the plugin by simple calling the method in the jQuery object

$('selector').strongPass();

And you can also customize the messages that apear for each case:

$('selector').strongPass({
    strong: 'Strong Password!',
    medium: 'Not so strong...',
    weak: 'You can\'t be serious',
});

By default the plugin will create a span to display the warnings, you can specify your own selector in the options: 

$('selector').strongPass({
    warning: '#pass-result'
});
